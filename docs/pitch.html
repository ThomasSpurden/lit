lit blurs the lines between source and documentation

Here is an overview of a hello world Haskell program. 
We define * as the macro from which all other code or macros exist.
    << table of contents >>=
    << print a string >>

Now our program prints a string to the world!
    << print a string >>=
    main = putStr "Hello, World!"

lit can process the literate file in several ways. To simply produce the code file run: $ lit -c helloWorld.hs.lit

Which generates the file helloWorld.hs

main = putStr "Hello, World!"
